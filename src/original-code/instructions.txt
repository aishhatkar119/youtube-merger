Unfortunately, there's no official way to merge YouTube accounts or transfer data between Google accounts. YouTube doesn't provide any built-in migration tool for this.

Here's what you can and can't transfer from a Google Takeout backup:

What you can manually migrate:

Subscriptions — Your takeout includes a subscriptions.csv file. You can use third-party tools like YouTube Subscriptions Transfer (a browser extension) to bulk-subscribe on your personal account.
Playlists — These are exported as .csv files. You can manually recreate them, or use tools/scripts to re-add videos via the YouTube Data API.
Watch history / Search history — These are exported as HTML/JSON files but cannot be imported into another account. You'd have to manually re-watch or there's no import path.
What cannot be transferred:

Channel (subscribers, videos, comments, analytics) — A YouTube channel is permanently tied to a Google account. There's no migration path.
Watch Later — Not included in Takeout in a useful import format.
YouTube Premium benefits — Non-transferable.
Best workaround for subscriptions (most practical):

Open your Takeout export and find subscriptions.csv
Install the Trim - IFTTT for YouTube extension or similar
Use it to import the subscription list to your personal account

**2. Set up Google Cloud credentials**

- Create a project → Enable **YouTube Data API v3**
- Go to **Credentials** → Create **OAuth 2.0 Client ID** (Desktop App type)
- Download the JSON and save it as `credentials.json` next to the script

**3. Set your Takeout path**
Edit the `CONFIG.takeoutPath` in the script to point to your Takeout folder. The expected structure is:

Takeout/YouTube and YouTube Music/
  subscriptions/subscriptions.csv
  playlists/my-playlist.csv

4. Run it (logged into your personal Google account)

bash
npx ts-node youtube-migrate.ts
It will open a URL — authorize it with your personal account, paste the code back, and it'll start migrating.

A few things to know:

The YouTube Data API has a daily quota of 10,000 units. If you have hundreds of subscriptions/videos, it may take 2–3 days to fully migrate across multiple runs (the script skips already-migrated items).
Liked videos are re-liked on your personal account (not added to a playlist).
Playlists are created as private by default — you can change them afterward.
